- **部署**：用户自有 VPS + Docker（1Panel 面板），或 Cloudflare Pages 均可，已提供 Dockerfile
- **形态**：纯静态 HTML/CSS/JS，无框架、无构建步骤

## 三、⚠ 素材的关键说明（重要，避免踩坑）

### 1. P01～P24 是**横版 16:9**，只有总览图是竖版

- 单页高清图：1672×941 PNG，接近 16:9
- 单页缩略图：800×450 WebP，标准 16:9
- `总览图`：800×1200 WebP，2:3 竖版

课程卡片应使用单列、图上文下的结构，并让图片区域保持 `aspect-ratio: 16 / 9`。不要再使用左图右文或把单页课件当成竖版素材。

### 2. 那份 MD 是**提示词，不是文案**
`备课\TJAD数字化课程_..._知识库.md` 是作者用来生成 PPT 的个人笔记，**不能直接照搬**。
当前网页文案已根据 PPT 实际内容 + 项目资料重写过一轮，但仍可优化。

### 3. 25 张图片的对应关系
- `总览图.png`：24 页 PPT 的拼图矩阵，用在 Hero 区
- `P01_知识的困境.png` ~ `P24_最终致谢.png`：每页对应一个内容卡片

文件名含中文，URL 需 UTF-8 编码（浏览器会自动处理，命令行测试要注意）。

## 五、本地预览方法

```bash
cd 网站
python -m http.server 8080 --bind 127.0.0.1
# 浏览器访问 http://127.0.0.1:8080
```

⚠ **Clash 代理提醒**：用户机器装了 Clash，可能拦截 localhost。若浏览器打不开：
- 用 `http://127.0.0.1:8080`（比 `localhost` 更可能直连）
- 或临时关闭 Clash 的「系统代理」/「TUN 模式」
- 命令行测试加 `--noproxy "*"` 参数
